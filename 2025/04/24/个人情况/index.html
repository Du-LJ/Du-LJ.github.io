个人情况

学习经历

2019年进入北京交通大学-交通运输学院-电子商务专业学习。

2023年以优异成绩保送研究生至本校本专业继续学习，并获北京交通大学学习一等奖学金。

预计2026年毕业。

实习经历

从2023至今，我辗转于不同的公司，不同的业务，不同的岗位，一方面是提升学习，另一方面，也是在不断地为自己的工作做准备。

在这里，我想特别感谢我的父亲，我的导师李老师，以及我的女朋友，没有他们的支持，我很难接触到这么多不同的岗位，也很难感受到不同公司的不同文化，不同城市的不同风采。

迪卡侬-中国电商-策略中台PMO实习生 上海 2025.02-至今

负责迪卡侬内部 OMS（订单管理系统）、WMS（物流管理系统）的系统维护、日常更新。

  • Bug 修复：搭建线上系统 Bugfix 价值体系，同步工单系统完成线上系统异常的修复。整理根源问题超过 20+，处理京东渠道的订单信息前后端同步问题，避免资损超 20w。

  • 订单拉回功能：对京东渠道订单拉回功能负责，了解 VIM 渠道模式，供应链体系，完成项目流程图设计，产出双系统更新 MRD、PRD，跟进开发流程。功能上线后，日拉回订单超过 100+，避免物流过程产生的资损日均 1w+。

项目管理：对电商平台升级改造内容负责，保证迪卡侬内部系统功能顺利上线。

  • JD 会员通 2.0：负责 ISV 服务商沟通，功能对比，输出内外部功能打通具体方案，项目仍在进行中。

  • 客服系统：对迪卡侬客服系统升级负责，梳理业务需求 120+条，撰写标书，参与招投标过程。

  • 物流 100：参与迪卡侬物流信息整合平台的升级项目，对接技术支持，实现供应商平台信息同步。

蔚来-内容平台营销部-产品经理实习生 上海 2024.09-2024.12

蔚来（内部）Library 产品：蔚来的内容、风控、营销信息阵地，以管线化信息处理为基础，接入 AI 实现应用功能。

大模型能力提升：针对 Library 应用场景，实现从 0 到 1 的大模型接入、训练、落地、优化。

  • AI 能力模块化：对市面上大模型 UI 进行调研，根据业务需求完成 Library 大模型搜索、组稿、竞品对比、舆论选题功能的 PRD、高保真原型图产出，Prompt 调优及 RAG 语料库优化。推动上线后，各功能使用 DAU 突破 500+。优化产品使用路径和逻辑。

  • 组稿能力：针对不同业务场景，将组稿功能作为外接大模型能力提升重点，设计产品信息 FABI 向量化逻辑，分析主流平台 200+爆款文稿，实现大模型组稿能力 0 到 1 突破。上线后一线销售部门使用 DAU 突破 800+，内容方向宣传部门使用 MAU 突破 2000+。

  • 模型调优：搜集产品运营日常热点问题 1W+，数据处理后进行人工标注，实现**准确率 78%到 92%**的提升。

Library 产品迭代：Library 上线后日常、月常迭代与功能更新。

  • 功能上线：全程参与 Library 首页的评审、排期、开发、测试、上线流程，测试中协助解决功能性 bug 与展示错误共 20+需求。

  • FABI 迭代：对产品亮点展示界面进行迭代，将原有的列表排列模式拓展为层级图模式，优化检索逻辑，提升每日平均 UV500+。

字节跳动-中国电商-平台大促组-产品运营实习生 上海 2024.04-2024.08

价格力产品：对平台商品价格进行准确的实时比价，解放价格审核工作量，结合黑白名单功能实现平台低价心智。

  • 价格力管控：提升抖音电商活动商品价格力优势，参与价格规则制定，设定巡检任务实时监控高价商品并议价或挂起，保障 520、618 等 SS 促活动周期活动池+频道池商品订单高价率≤5% ，对电商大盘非高价订单量贡献超 35%。

  • 货品运营：为保障活动商品池优质货品供应，算法圈选等方式保障活动商品供给，设定不同商品比价优先级，520 期间达成圈选商品报名率 85%，618 期间达成报名率 80%。618 完成商品活动报名超 1000W，为大盘 GMV 贡献超 25%。

  • 搜索产品策略：帮助频道打造低价鲜花心智，实现内容场与货架场策略打通，在抖音主端+极速版上线搜索侧资源，提升“鲜花好礼”搜索词 PV 超 1200W，优先坑位选品总 GMV 超 500W。对专项整体 PV 贡献超 40%、对超值购频道 GMV 贡献超 30%。

  • B 端招商策略：改善 B 端触达资源难以管理的情况，设立活动招商触达 SOP，通过 GMV 或订单量导向，划分商家层级，通过触达产品功能实现对不同层级商家的不同透传策略。针对 S4+商家透传有效率超过 79%。实现有效报名超千万商品。

元气森林-零售业务中心-用户/渠道运营实习生 北京 2023.03-2023.06

  • 渠道运营：选品匹配视频、音乐、社区等渠道需求，与渠道供应商沟通确认商品投放方式、流量玩法，监督投放效益及各类数据。商品曝光覆盖头部 APP，私域粉丝日均增长 2K+。

  • 店铺运营：管理微信及支付宝小程序商城，完成页面设计，推广规划。通过多渠道投放、低价小爆品、日常活动等方式完成流量的盘活，关注时令热点，推出季节款爆品，MAU 稳定增长，月 GMV 提升超 20W。

  • 临期品管理：跨部门协调临期品库存，实现临期货品的销路打通。两店铺临期商品 PV 日均 5~8W，GMV 日均提升 2~3W。

乐队经历

在社团中的时光一直是我难以忘记的部分。无论是社团的每一次活动，还是我们自己的乐队——坦克抛锚。

说到这个名字，有一个好玩的故事：时任吉他手“繁星”（化名）与时任键盘手LX同学住隔壁宿舍，一天正逢天降大雨，寝室阿姨拿出了大喇叭疏导同学们，LX同学突发奇想，偷拿了阿姨的喇叭，在楼道中以魔幻的嗓音，喊出了：“繁星，繁星，你坦克抛锚了。” 于是，便有了坦克抛锚。

从大一起，除了上课、睡觉、打游戏、吃饭之外，我更多的时间是与这几位艺术家们辗转于各种live house，翻弹我们喜欢的乐队的歌曲。

虽然大家都已各奔东西，我也来到上海将近一年，但坦克都已经抛锚了，挪也挪不动。

感谢坦克抛锚的各位。

社团经历

  • 主要负责学院内部组织的文艺活动和大型演出。
  • 独立完成活动策划、内容编排和现场指挥工作。

LAMP音乐联盟 会长 2022.06-2023.12

  • 策划校内外大型演出，与新浪微博合作草地音乐节活动。
  • 负责完成学校演出任务，支持学校文艺活动工作。

在LAMP的时光，遇到了对我帮助最大的学长，遇到了一群志同道合的伙伴，我们完成了一些很牛逼的工作。

感谢LAMP，让音乐充实我的生活。

游戏经历

游戏习惯 ：3A游戏，打牌爬塔类游戏爱好者，热衷于FPS、Rogue-like类独立游戏。每年平均3~4款头部游戏深度体验，体验时间均1月以上。

网络游戏：金铲铲之战900小时（最高钻石III）；瓦洛兰特400+小时，充值2000+；英雄联盟1200+小时，充值3000+。炉石传说标准、狂野双传说，酒馆战棋9000分。逃离塔克夫黑边玩家，超500小时。守望先锋开服玩家（已弃坑）。

单机游戏：目前Steam账号价值11328元，游戏时间超3000小时，共180+游戏。常玩：博德之门3、无人升空、双人成行、赛博朋克2077、星露谷物语、骰子浪游者、小丑牌、杀戮尖塔、以撒的结合：重生、Hades等。

我个人是非常喜欢游戏、热爱游戏的，从小时候接触PSP游戏，到现在各种平台各类型游戏，无论3A大作还是独立游戏，单机或网络游戏，我都希望有一天能够作为一个游戏行业的从业者，实现从玩家到设计者的转变。

博客作用

说到这个博客，首先会作为我的文章发布平台，也会作为我学习记录的一个小网盘。

我会将自己的学习心得，对游戏的拆解，编辑成一篇篇文章，做不定时的更新。

当然不止for找工作用，而是将作为我的一个宝贵财富。
